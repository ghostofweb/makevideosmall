# Security Policy

## Supported Versions
Only the latest stable release of Make Video Small is supported for security updates.

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0.0 | :x:                |

## Reporting a Vulnerability
If you discover a security vulnerability within this project, please **do not** open a public issue. Instead, please report it privately to the maintainer:

* **Contact:** Reach out to Sahiljeet Singh Kalsi via GitHub profile or personal website (ghostofweb.vercel.app).
